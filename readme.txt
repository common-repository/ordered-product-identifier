Grow your customer's trust on delivered product by uniquely identify ordered products.

== Description ==

This plugin facilitate your customer to assure the product arrived is the same that has been delivered. It provides you an extra feature of adding ordered product identification number for products which are ordered and it will be displayed on front-end order detail page when order is completed.

= Main features =

* **Provide an extra feature of providing unique identification number to ordered products.**
* Unique identification number is also shown on front-end order detail page when order is completed.
* **It facilitates you to identify ordered products.**
* You can choose in between manually or auto generation of unique identification number.
* You can use any combination for generating unique identification number.
* You can also use barcodes as unique identification number.
* You can get set Prefix in unique identification number.
* You can keep unique identification number for product history.
* You can use it when there is any return request for products.
* You can use during dealing with damaged products.
* Compatible with invoice plugins like **Woocommerce pdf Invoices Packing Slips**, **WooCommerce Print Invoice & Delivery Note**
* Easy to configure

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of the plugin, search the plugin download it and add this plugin through add new section.

In the search field type “Ordered Product Identifier” and click Search Plugins. Once you’ve found our eCommerce plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
